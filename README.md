# House Price Prediction

A simple Machine Learning web application built with Python and Streamlit to predict house prices.

## Live Demo
🚀https://house-price-prediction-nxvdpmxqbscduinuhrnn3f.streamlit.app/

## 📌 Features
- Predict house prices instantly
- User-friendly Streamlit interface
- Machine Learning model using Scikit-learn
- Real-time prediction based on input values

## Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Joblib

- ## 📈 Model
- Algorithm: Linear Regression
- Framework: Scikit-learn
- Model saved using Joblib
